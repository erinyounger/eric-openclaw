# ToDoList 应用规格说明书

## 1. 项目概述

- **项目名称：** EricTodo
- **项目类型：** Web 应用 (Next.js)
- **核心功能：** 简洁优雅的待办事项管理工具
- **目标用户：** 追求效率与美感的用户

## 2. UI/UX 规格

### 布局结构

- **整体布局：** 居中卡片设计，最大宽度 480px
- **页面结构：**
  - 顶部：应用标题 + 简洁装饰
  - 中央：输入区域 + 任务列表
  - 底部：统计信息

### 响应式断点

- Mobile: < 640px（全宽度 padding: 16px）
- Desktop: >= 640px（居中卡片）

### 视觉设计

**色彩方案（暗色主题）：**
- 背景: `#0f0f0f` (深黑)
- 卡片背景: `#1a1a1a` (炭灰)
- 主色调: `#22d3ee` (青色 cyan-400)
- 辅助色: `#a78bfa` (紫色 violet-400)
- 文字主色: `#fafafa` (zinc-50)
- 文字次色: `#a1a1aa` (zinc-400)
- 边框: `#27272a` (zinc-800)
- 完成状态: `#4ade80` (green-400)

**字体：**
- 标题: `Geist Sans` 或 `Inter`, 24px, font-weight: 700
- 正文: `Geist Sans` 或 `Inter`, 16px
- 任务文字: 15px

**间距系统：**
- 卡片内边距: 24px
- 任务项间距: 12px
- 输入框高度: 48px

**视觉效果：**
- 卡片圆角: 16px
- 输入框/按钮圆角: 12px
- 轻微阴影: `0 4px 24px rgba(0,0,0,0.3)`
- 完成任务 strikethrough 效果 + 透明度降低

### 组件列表

1. **Header** - 应用标题 + 装饰性图标
2. **TodoInput** - 输入框 + 添加按钮
   - 输入框：占位符 "添加新任务..."
   - 按钮：圆形，加号图标，青色背景
3. **TodoList** - 任务列表容器
4. **TodoItem** - 单个任务项
   - 复选框（自定义样式）
   - 任务文字（可编辑）
   - 删除按钮（悬停显示）
5. **TodoFilter** - 筛选标签（全部/进行中/已完成）
6. **TodoStats** - 底部统计（剩余任务数）

### 交互行为

- **添加任务：** 输入框回车或点击按钮
- **完成任务：** 点击复选框，文字划线 + 淡色
- **删除任务：** 悬停显示删除按钮，点击删除
- **筛选：** 点击标签切换，显示对应任务
- **动画：** 任务添加/删除时淡入淡出

## 3. 功能规格

### 核心功能

1. **添加任务** - 输入文字，按回车或点击按钮添加
2. **标记完成** - 点击复选框切换完成状态
3. **删除任务** - 点击删除按钮移除任务
4. **筛选查看** - 全部 / 进行中 / 已完成
5. **数据持久化** - 使用 localStorage 保存

### 数据结构

```typescript
interface Todo {
  id: string;
  text: string;
  completed: boolean;
  createdAt: number;
}
```

### 边界情况

- 空输入时不添加任务
- 防止 XSS（React 默认防护）
- localStorage 不可用时内存暂存
